Chapter 4. Working with Strings

字符类型：

    • binary strings
    • BINARY
    • VARBINARY
    • BLOB
    • nonbinary strings
    • CHAR
    • VARCHAR
    • TEXT

查看nobinary 字符集

  • mysql> SHOW CHARACTER SET;

查看string 占用字节数

  • mysql> SET @s = CONVERT('abc' USING utf8);
  • mysql> SELECT LENGTH(@s), CHAR_LENGTH(@s);
  • +------------+-----------------+
  • | LENGTH(@s) | CHAR_LENGTH(@s) |
  • +------------+-----------------+
  • | 3 | 3 |
  • +------------+-----------------+
  •  
  • mysql> SET @s = CONVERT('abc' USING ucs2);
  • mysql> SELECT LENGTH(@s), CHAR_LENGTH(@s);
  • +------------+-----------------+
  • | LENGTH(@s) | CHAR_LENGTH(@s) |
  • +------------+-----------------+
  • | 6 | 3 |
  • +------------+-----------------+
