Centos7 安装nfs

Server 端操作

yum -y install nfs-utils rpcbind

/etc/exports

/production/attachment/ 192.168.7.4(rw,no_all_squash,anonuid=1000,anongid=1000)

启动nfs

systemctl start rpcbind

systemctl start nfs

systemctl enable rpcbind

systemctl enable nfs

Client 端操作

yum -y install nfs-utils rpcbind

systemctl start rpcbind

systemctl enable rpcbind

然后就可以mount 了
